# Smart Receipt Project

An AI-powered receipt and expense tracking application.

## Main Technology Plan
- **Backend**: Python with FastAPI
- **Database**: MySQL
- **Database ORM**: SQLAlchemy
- **Database Migrations**: Alembic
- **AI Service**: Gemini API
- **Excel Export**: Pandas and OpenPyXL
- **Frontend**: To be decided (Streamlit or React with TypeScript)

## Current Status
This repository currently contains only the initial clean folder structure and template configuration files.
No database, AI service, business logic, or frontend UI has been implemented yet.
