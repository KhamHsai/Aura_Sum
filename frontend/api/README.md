# Frontend API Integration

This folder will hold the API client setup for calling backend endpoints, including:
- GET `/api/health`
- POST `/api/receipts/analyze`
- POST `/api/expenses`
- GET `/api/expenses`
- GET `/api/expenses/{id}`
- PUT `/api/expenses/{id}`
- DELETE `/api/expenses/{id}`
- GET `/api/categories`
- GET `/api/reports/monthly`
- GET `/api/reports/monthly/export`
